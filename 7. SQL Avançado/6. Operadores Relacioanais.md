# Possíveis operadores

= - Igual a

like - Inclui, pode-se usar % como sufixo ou prefixo para find, aqui, mesmo para campos do tipo number, utilizar as aspas simples para passar o dado procurado

> - Maior que

< - Menor que

>= Maior ou igual

<= Menor ou igual

<> Não igual a

!= Diferente de
